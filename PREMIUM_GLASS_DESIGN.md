## üé® Glass Design Principles

### Transparency Levels
- **Heavy Glass (70-80%)**: Navigation, major sections, CTAs
- **Medium Glass (60-70%)**: Cards, panels, forms
- **Light Glass (50-60%)**: Feature cards, info panels
- **Subtle Glass (40-50%)**: Add-ons, minor elements

### Blur Intensity
- **Maximum Blur (40px)**: Navigation, CTAs, major sections
- **Strong Blur (30px)**: Product cards, viewer, details
- **Medium Blur (20px)**: Feature cards, add-ons, buttons
- **Light Blur (10px)**: Subtle effects

### Saturation Enhancement
- **120%**: Standard for most glass elements
- **150%**: Premium sections (CTAs, achievements)
- **180%**: Navigation for maximum vibrancy

## üíé Premium Shadow System

### Multi-Layer Shadows
Every glass element uses **dual-layer shadows**:

1. **Outer Shadow**: Depth and elevation
   ```css
   0 8px 32px rgba(0, 0, 0, 0.06)
   ```

2. **Inner Glow**: Glass edge highlight
   ```css
   0 1px 0 0 rgba(255, 255, 255, 0.8) inset
   ```

### Shadow Intensities
- **Subtle**: 0.04 - 0.06 opacity
- **Medium**: 0.08 - 0.10 opacity
- **Strong**: 0.12 - 0.14 opacity

### Hover Enhancements
On hover, cards get:
- **Deeper shadows**: 2-3x base intensity
- **Brighter inner glow**: 0.9 - 1.0 opacity
- **Increased elevation**: -4px to -6px transform
- **Enhanced opacity**: +10-15% transparency

### Accessibility
- **Contrast maintained**: Glass opacity levels ensure readability
- **Text clarity**: All text on solid or semi-solid backgrounds
- **Focus visible**: Clear focus indicators on all interactive glass
- **Reduced motion**: Honors prefers-reduced-motion
